# ALL SAFE — Setup Guide

## Prerequisites
- Python 3.9+
- Node.js 18+

## Backend (FastAPI)
```powershell
cd backend
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
Backend: http://localhost:8000

## Frontend (React + Vite)
```powershell
cd frontend
npm run dev
```
Frontend: http://localhost:5173
